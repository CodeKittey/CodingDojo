Die Aufgabe:

Implementiere eine Web-API (REST-Service) mit folgenden Schnittstellen:
  • GET /onwerkstaff
  • GET /onwerkstaff/:employeeId
  • PUT /onwerkstaff

GET /onwerkstaff

Liefert ein JSON-Array: 
                  [
                    {
                      "name" : "Mehl",
                      "prename" : "Jan",
                      "codinglanguages" : ["php", "C#", ...]
                    },
                    ....
                  ]

GET /onwerkstaff/:employeeId

Liefert ein JSON-Objekt: 
                    {
                      "name" : "Unicorn",
                      "prename" : "Jens",

                    }
